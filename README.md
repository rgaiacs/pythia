# Pythia

## Install

### For developers

```
$ python -m pip install -e .
```

### For users

```
$ python -m pip install .
```

## Test

Run

```
$ python -m unittest
```

## Use (from Command Line)

```
$ pythia --help
```

## Use (as Library)

```
import pythia
```

## Benchmark (with Neptune)

```
$ pythia2neptune --help
```
